The Radar Map provides a multi-dimensional visualization of key financial metrics for companies listed in the S&P 500, based on data from the Yahoo API. A dynamic filter allows users to select multiple companies for side-by-side comparison, with hues and scales adjusting automatically for clarity. Tooltips appear on each financial attribute when hovered over, offering immediate context and additional information for each metric.
